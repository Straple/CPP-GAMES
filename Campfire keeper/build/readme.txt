builds:

Release: обычная игра
Development: добавляет возможность видеть информацию для разработчиков(чекните, если интересно)

install:

Если не запускается Game Engine.exe
тогда нужны доп. библиотеки, которые можно скачать с офф. сайта
https://support.microsoft.com/en-us/topic/the-latest-supported-visual-c-downloads-2647da03-1eea-4433-9aff-95f26a218cc0
или запустить уже скачанные VC_redist нужной вам платформы

run:

single запускает соло режим
server запускает сервер для мультиплеера
client запускает клиента для мультиплеера, который будет подключаться к серверу

используйте ip 127.0.0.1 для локальной игры


game:

WASD ходьба
SPACE прыжок
J удар
ESC выход
ENTER сменяет полноэкранный режим на оконный и наоборот

для development build также доступны:
TAB сменяет режим отладки
K показывает круги симуляции врагов
F показывает fps, время между кадрами в ms, все время игры
UP/DOWN изменяет режим масштабирования рендеринга (будете больше/меньше видеть)


механика игры:

сложно объяснить. много всего
